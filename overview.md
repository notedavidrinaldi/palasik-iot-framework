# PALASIK Project Overview

## 1. Latar Belakang
Perkembangan Internet of Things (IoT) membawa tantangan besar pada aspek keamanan, khususnya pada perangkat edge seperti Raspberry Pi yang sering digunakan tanpa mekanisme keamanan berlapis.

Sebagian besar sistem IoT masih mengandalkan model keamanan berbasis perimeter, yang tidak lagi relevan pada lingkungan terdistribusi.

## 2. Masalah Penelitian
Beberapa permasalahan utama pada IoT Edge:
- Device dipercaya secara default
- Tidak ada evaluasi perilaku berkelanjutan
- Lemah terhadap spoofing & rogue device
- Minim framework riset berbasis edge

## 3. Tujuan PALASIK
PALASIK dikembangkan untuk:
- Menerapkan **Zero Trust Model pada IoT Edge**
- Menghitung *trust score* perangkat secara dinamis
- Menjadi framework riset terbuka berbasis Raspberry Pi

## 4. Kontribusi Ilmiah
Kontribusi utama PALASIK:
1. Trust Engine berbasis Edge
2. Behavioral-based IoT Security
3. Modular framework untuk riset kampus
4. Dataset & flow Node-RED reproducible

## 5. Target Pengguna
- Mahasiswa (skripsi / tesis)
- Dosen & peneliti
- Praktisi IoT & keamanan
